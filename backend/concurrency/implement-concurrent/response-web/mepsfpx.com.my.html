PLEASE NOTE
For improved security, usage of FPX requires that you resolve the domain mepsfpx.com.my and paynet.my using public DNS. Please ensure that your servers do not statically refer to the mepsfpx.com.my via a hosts file.
 
HARAP MAKLUM
Sebagai salah satu langkah terhadap keselamatan maklumat yang lebih kukuh, penghalaan data kepada FPX memerlukan resolusi DNS. Sila pastikan anda tidak menggunakan fail hosts.
